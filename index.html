api.apiembed.com
The API root, available in both HTTP and HTTPS flavors.
source
Source URL, a HAR Request object as the http request description format
targets
An ordered comma separated list of all the desired targets, current available values: all
Note:

The order of the targets list will determine the tab output order of the widget
